投稿フォーム


ページ全体 src/app/content/contribution

webの標準的な入力フォームに Angular の動的制御を組み込んだサンプル。

フォーム送信（API通信）前のフールプルーフとして正規表現や文字数制約のチェックを入力したタイミングで実施する。 その入力制約状態を元にメッセージの表示やログインボタンを無効化する。

入力部品 src/app/ui/label-input

label要素、input要素、エラーメッセージをまとめたコンポーネント。

コンポーネントとは、一連の UI 機能を再利用できるようにするための UI 内の責任領域を定義する Angular の基本機能。 1UI 要素として HTML および Typescript により実装する。

可変部分はパラメータとして指定することで、HTML 構成の実装を省力化やメンテナンス性向上を図る。
